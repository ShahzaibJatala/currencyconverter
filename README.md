# 💱 Real-Time Currency Converter

A dynamic and responsive web application that provides real-time currency exchange rates. This project highlights my transition into intermediate JavaScript development, specifically focusing on fetching data from external APIs and handling asynchronous operations.

## Features

* **Live Exchange Rates:** Fetches the most up-to-date currency data using the **Fetch API**.
* **Global Currency Support:** Supports conversion between a wide variety of global currencies.
* **Dynamic Flag Updates:** Automatically updates country flags based on the selected currency.
* **Responsive Design:** A clean, mobile-friendly interface built with modern CSS techniques.
* **Error Handling:** Robust logic to handle API failures or empty input values.
